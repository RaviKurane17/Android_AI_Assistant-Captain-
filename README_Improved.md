# 📖 Overview

Captain AI Phone Agent is a next-generation Android AI assistant that continuously listens for an offline wake word, understands natural language, and performs intelligent actions on both Android and Windows.

Unlike traditional assistants, Captain combines:

- 🎙 Offline Wake Word Detection
- 🧠 Groq Powered AI
- 📱 Android Device Automation
- 💻 Windows PC Remote Control
- ☁ Firebase Real-time Sync
- 🔐 Secure API Key Storage

# 🚀 Getting Started

## 1. Clone

```bash
git clone https://github.com/<YOUR_USERNAME>/captain-ai-phone-agent.git
```

## 2. Configure Firebase

- Create Firebase Project
- Enable Realtime Database
- Add google-services.json

## 3. Add API Keys

- Groq API
- Picovoice Access Key

## 4. Build

```bash
./gradlew assembleDebug
```
